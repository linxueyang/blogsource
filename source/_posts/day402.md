    - git 大文件处理

  ```
  1. 安装GitHub LFS
  2. git lfs install
  3. git lfs track "*.bz2" 某些后缀的大文件
  4. git add .gitattributes
  5. git add xxx
  6. git commit -m
  7. git push
  ```

  > 最好关闭防火墙
  > 只有先把".gitattributes"传上去，才可以上传大文件。
